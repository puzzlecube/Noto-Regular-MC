Google Noto-Regular fonts for Minecraft (1.16.X)

Google's Noto fonts can be downloaded from https://www.google.com/get/noto/
Google's Noto fonts are distributed under the SIL Open Font License v1.1
	See ./assets/minecraft/font/OFL.txt for the license text.

./assets/minecraft/font/indexfont.js is licensed under a 2-clause BSD license.

You can run the script by changing directory to ./assets/minecraft/font/ and running:
	node indexfont.js
This will read all TTF and OTF files contained in the directory and produce the default.json needed for MC to load them all
You can add your own fonts to the font directory and run this script to add/replace fonts in this package
